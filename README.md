# IT Support Portfolio

A personal IT Support / Help Desk portfolio website built with HTML, CSS, and JavaScript.

## Overview

This project presents:
- Professional summary and career objective
- Technical skills grouped by category
- Troubleshooting workflow and tools
- IT project highlights with architecture/workflow diagrams
- Lab environment section and contact information

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- SVG diagrams
- Font Awesome icons

## Run Locally

Because this is a static site, you can open it directly in your browser:

1. Clone the repository
2. Open `hdp.html`

Optional (recommended), run a local server:

```bash
# Python 3
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).
